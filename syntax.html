PHP Syntax Reference

1. PHP Tags

PHP code is embedded inside <?php ... ?> tags.

<?php
// PHP code goes here
echo "Hello, world!";
?>

2. Comments

  • Single-line: // or #
  • Multi-line: /* ... */
// Single-line comment
# Another single-line comment
/* Multi-line
   comment */

3. Variables

Variables start with $ and are case-sensitive.

$name = "Alice";
$age = 25;
$pi = 3.14;

4. Data Types

  • String
  • Integer
  • Float (double)
  • Boolean
  • Array
  • Object
  • NULL
$str = "Hello";
$int = 10;
$float = 10.5;
$bool = true;
$arr = array("apple", "banana");
$null = NULL;

5. Strings

Use single or double quotes; double quotes parse variables.

$greeting = "Hello, $name!";
$msg = 'Hello, $name'; // Outputs literal $name

6. Constants

Defined with define() or const.

define("SITE_NAME", "MyWebsite");
const MAX_SIZE = 100;

7. Operators

  • Arithmetic: + - * / % **
  • Assignment: = += -= *= /= %=
  • Comparison: == === != !== < > <= >=
  • Logical: && || !
$sum = 5 + 3;
$check = (5 > 3) && (2 < 4);

8. Conditional Statements

if ($age >= 18) {
    echo "Adult";
} elseif ($age >= 13) {
    echo "Teen";
} else {
    echo "Child";
}

9. Switch Statement

switch ($color) {
    case "red":
        echo "Red color";
        break;
    case "blue":
        echo "Blue color";
        break;
    default:
        echo "Other color";
}

10. Loops

for loop

for ($i = 0; $i < 5; $i++) {
    echo $i;
}

while loop

$i = 0;
while ($i < 5) {
    echo $i;
    $i++;
}

foreach loop

$fruits = array("apple", "banana");
foreach ($fruits as $fruit) {
    echo $fruit;
}

11. Functions

function add($a, $b) {
    return $a + $b;
}
$sum = add(3, 4);

12. Arrays

Indexed Arrays

$cars = array("Volvo", "BMW", "Toyota");
echo $cars[0];

Associative Arrays

$age = array("Peter"=>35, "Ben"=>37, "Joe"=>43);
echo $age["Peter"];

13. Superglobals

  • $GLOBALS
  • $_SERVER
  • $_REQUEST
  • $_POST
  • $_GET
  • $_SESSION
  • $_COOKIE
  • $_FILES
  • $_ENV
echo $_SERVER["PHP_SELF"];

14. include & require

  • include produces warning and continues
  • require produces fatal error and stops
include "header.php";
require "config.php";

15. Error Handling

try {
    if ($file === false) {
        throw new Exception("File not found.");
    }
} catch (Exception $e) {
    echo "Error: " . $e->getMessage();
}
