HTML Elements

Deep dive into every essential HTML element with syntax, semantics, and live examples.

<!DOCTYPE html> & <html>

Every modern HTML file begins with a document-type declaration followed by the <html> root element.

<!DOCTYPE html>
<html lang="en">
  ...
</html>

<head> Elements

  • <meta charset="UTF-8"> – declares character encoding.
  • <meta name="viewport"> – responsive layout control.
  • <title> – sets the browser tab title.
  • <link rel="stylesheet"> – external CSS.
  • <style> – embedded CSS.
  • <script> – JavaScript (defer, async, module).
  • <base href="/"> – base URL for relative links.

Sectioning Elements

  • <body> – document body.
  • <header> – introductory content.
  • <nav> – navigation links.
  • <main> – dominant page content (only one per doc).
  • <section> – thematic grouping.
  • <article> – self-contained composition.
  • <aside> – tangential content (sidebar).
  • <footer> – footer info.

Text-Level Semantics

  • <h1>-<h6> – six heading levels.
  • <p> – paragraph.
  • <a href> – hyperlink.
  • <strong> – strong importance.
  • <em> – stress emphasis.
  • <mark> – highlighted text.
  • <small> – fine print.
  • <span> – generic inline container.
  • <br> – line break.
  • <hr> – thematic break.

List Elements

  • <ul> – unordered list.
  • <ol> – ordered list (type, start, reversed).
  • <li> – list item.
  • <dl> – description list.
  • <dt> – description term.
  • <dd> – description details.

Embedded Content

  • <img src alt> – image.
  • <picture> – responsive images with sources.
  • <source> – media resource for picture/video/audio.
  • <video src controls> – video player.
  • <audio src controls> – audio player.
  • <track kind="captions"> – subtitles/captions.
  • <iframe src> – nested browsing context.
  • <embed src> – external integration.
  • <object data> – generic external resource.

Table Elements

  • <table> – table wrapper.
  • <caption> – table title.
  • <thead> – header block.
  • <tbody> – body block.
  • <tfoot> – footer block.
  • <tr> – table row.
  • <th scope> – header cell.
  • <td> – data cell.
  • <col> & <colgroup> – column styling.

Form Elements

  • <form action method> – form container.
  • <label for> – caption for control.
  • <input type> – text, email, password, checkbox, radio, range, color, date, file, hidden, submit, …
  • <textarea> – multi-line text.
  • <select> & <option> – dropdown.
  • <optgroup label> – grouped options.
  • <button type> – clickable button (submit/reset/button).
  • <fieldset> & <legend> – grouped controls.
  • <datalist id> – predefined suggestions for input.
  • <output for> – calculated result.
  • <progress value max> – progress bar.
  • <meter value min max> – scalar gauge.

Interactive Elements

  • <details> – disclosure widget.
  • <summary> – summary/legend for details.
  • <dialog open> – modal or popup box.

Scripting & Data

  • <script type="module"> – JavaScript.
  • <noscript> – fallback when JS disabled.
  • <template> – inert chunk of DOM for cloning.
  • <slot> – shadow DOM placeholder.
  • <canvas> – 2D/3D drawing surface.
  • <svg> – inline scalable vector graphics.
  • <math> – MathML for equations.

Obsolete / Deprecated (Avoid)

<font>, <center>, <marquee>, <blink>, <frameset>, <big>, <strike> — use CSS instead.
