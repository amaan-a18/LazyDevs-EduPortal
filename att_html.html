HTML Attributes

accesskey

Global

Provides a keyboard shortcut to focus an element. Combine with Alt on Windows or Control+Alt on Mac.

<button accesskey="s">Save</button>

class

Global

Assigns one or more CSS classes to an element for styling or JavaScript selection.

<div class="card highlight">…</div>

contenteditable

Global

Makes element content editable by users. Values: true | false.

<p contenteditable="true">Edit me!</p>

data-*

Global

Stores custom private data for scripting. Replace * with any valid name.

<li data-id="42" data-user="ada">…</li>

dir

Global

Sets text direction: ltr, rtl, or auto.

<span dir="rtl">مرحبا</span>

draggable

Global

Specifies whether element can be dragged. Values: true | false.

<img draggable="false" src="logo.svg" alt="Logo">

hidden

Global

Hides element from all presentations. Boolean attribute.

<div hidden>Loading…</div>

id

Global

Unique identifier for element. Must be unique in entire document.

<section id="pricing">…</section>

lang

Global

Declares language of element content using BCP 47 codes.

<html lang="en-US">…</html>

spellcheck

Global

Hints browser whether to check spelling. Values: true | false.

<textarea spellcheck="true"></textarea>

style

Global

Inline CSS declarations for element.

<div style="color:lime; margin:1rem">…</div>

tabindex

Global

Controls tab order. 0 = in order, >0 = before, −1 = unreachable.

<button tabindex="0">Focusable</button>

title

Global

Advisory information shown as tooltip on hover.

<abbr title="HyperText Markup Language">HTML</abbr>

accept

<input type="file">

Comma-separated list of allowed file types or MIME patterns.

<input type="file" accept="image/png,image/jpeg">

alt

<img>, <area>, <input type="image">

Alternative text for images when unavailable.

<img src="cat.jpg" alt="Sleeping cat">

autocomplete

<form>, <input>, <select>, <textarea>

Controls browser autofill. Values: on | off or detailed tokens.

<input name="email" autocomplete="email">

autofocus

Form controls

Focuses element automatically on page load. Boolean.

<input autofocus>

checked

<input type="checkbox|radio">

Pre-selects input. Boolean.

<input type="checkbox" checked>

disabled

Form controls, <fieldset>, <optgroup>, <option>

Disables interaction. Boolean.

<button disabled>Submit</button>

for

<label>, <output>

Associates label with form control whose id equals this value.

<label for="email">Email</label>

form

Form controls

Overrides parent form; value is id of form to submit with.

<input form="signup" name="extra">

max / min

Range inputs, <meter>, <progress>

Upper/lower bounds for numeric or date inputs.

<input type="date" min="2025-01-01" max="2025-12-31">

maxlength

<input>, <textarea>

Maximum number of characters user may enter.

<input maxlength="10">

multiple

<input>, <select>

Allows multiple values. Boolean.

<input type="file" multiple>

name

Form controls

Key under which control value is submitted.

<input name="username">

pattern

<input>

Regular expression value must match for validity.

<input pattern="\d{4}" title="4 digits">

placeholder

<input>, <textarea>

Short hint displayed when field is empty.

<input placeholder="Search…">

readonly

<input>, <textarea>

Makes control non-editable but still focusable and submitted.

<input readonly value="Read">

required

Form controls

Element must be filled before form submission. Boolean.

<input required>

step

Numeric/date inputs

Granularity of value increments (e.g., 1, 0.1, 60).

<input type="number" step="0.01">

type

<input>, <button>

Determines control behavior or input subtype.

<input type="range">

value

Form controls, <li>, <meter>, <progress>

Current data or initial value.

<input value="default">

src

<img>, <video>, <audio>, <script>, <iframe>, <source>

URL of resource to embed.

<img src="hero.jpg" alt="Hero">

width / height

Media elements

Dimensions in CSS pixels (no units).

<canvas width="800" height="400"></canvas>

controls

<video>, <audio>

Shows playback controls. Boolean.

<video src="vid.mp4" controls></video>

loop

<video>, <audio>

Restarts media when finished. Boolean.

<audio loop></audio>

muted

<video>, <audio>

Silences audio output. Boolean.

<video muted></video>

preload

<video>, <audio>

Hints how much to buffer: none, metadata, auto.

<audio preload="metadata"></audio>

poster

<video>

Image shown until video plays.

<video poster="thumb.jpg"></video>

href

<a>, <area>, <link>, <base>

Target URL or resource path.

<a href="/about">About</a>

target

<a>, <area>, <form>

Where to open link: _self, _blank, _parent, _top, or frame name.

<a href="new" target="_blank">Open new tab</a>

download

<a>, <area>

Prompts user to download resource instead of navigating.

<a href="file.pdf" download>Save PDF</a>

rel

<a>, <area>, <link>, <form>

Relationship of target to current doc (e.g., nofollow, external, help).

<link rel="stylesheet" href="main.css">

colspan / rowspan

<td>, <th>

Number of columns/rows cell spans.

<td colspan="2">Merged</td>

scope

<th>

Defines cells header relates to: row, col, rowgroup, colgroup.

<th scope="row">Total</th>

charset

<meta>

Character encoding declaration.

<meta charset="UTF-8">

name

<meta>, <param>

Metadata key when used with content.

<meta name="viewport" content="width=device-width">

content

<meta>

Value associated with name or http-equiv.

<meta name="author" content="Ada">

http-equiv

<meta>

Pragma directive simulating HTTP header.

<meta http-equiv="refresh" content="5">

defer

<script>

Delays script execution until HTML parsing complete. Boolean.

<script src="app.js" defer></script>

async

<script>

Downloads script asynchronously and executes ASAP. Boolean.

<script async src="analytics.js"></script>

open

<details>

Expands disclosure by default. Boolean.

<details open><summary>More</summary>…</details>

start

<ol>

Initial number for list items.

<ol start="10">…</ol>

reversed

<ol>

Reverses numbering direction. Boolean.

<ol reversed>…</ol>

selected

<option>

Pre-selects option. Boolean.

<option selected>Default</option>

wrap

<textarea>

How text wraps on submission: soft (default) or hard.

<textarea wrap="hard"></textarea>
