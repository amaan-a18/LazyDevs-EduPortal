PHP Variables – Complete Guide

1. What is a Variable?

A variable in PHP is a name given to a memory location that holds data. Variables start with a dollar sign $ followed by the name.

<?php
$name = "Alice";
$age  = 25;
echo $name; // Outputs: Alice
?>

2. Rules for PHP Variables

  • Start with $ symbol.
  • Cannot begin with a number.
  • Only contain alphanumeric characters and underscores A-z, 0-9, _.
  • Case-sensitive ($name ≠ $Name).

3. Variable Scope

Local Scope

<?php
function test() {
    $x = 5; // local
    echo $x;
}
test(); // 5
?>

Global Scope

<?php
$x = 10; // global
function test() {
    global $x;
    echo $x;
}
test(); // 10
?>

Static Variables

<?php
function counter() {
    static $c = 0;
    $c++;
    echo $c;
}
counter(); // 1
counter(); // 2
?>

4. Variable Variables

A variable whose name is stored in another variable.

<?php
$a = "hello";
$$a = "world";
echo $hello; // Outputs: world
?>

5. Superglobals

Built-in arrays accessible everywhere:

  • $_GET
  • $_POST
  • $_REQUEST
  • $_SERVER
  • $_SESSION
  • $_COOKIE
  • $_FILES
  • $_ENV
<?php
echo $_SERVER["PHP_SELF"]; // current script path
?>

6. Data Types & Type Juggling

<?php
$int   = 10;
$float = 10.5;
$str   = "100";
$sum   = $int + $str; // $sum is 110 (int)
var_dump($sum);
?>
PHP automatically converts types based on context.

7. Constants vs Variables

<?php
define("SITE_NAME", "MySite");
echo SITE_NAME; // MySite

// Unlike variables, constants are global and case-sensitive by default.
?>

8. Best Practices

  • Use meaningful names: $userCount instead of $n.
  • Stick to lowercase with camelCase or underscores.
  • Initialize variables before use to avoid notices.
  • Avoid global variables; prefer passing parameters or using dependency injection.
