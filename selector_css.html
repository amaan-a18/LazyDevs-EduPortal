CSS Selectors - Complete Guide

What are CSS Selectors?

CSS selectors are used to select the HTML elements you want to style. They are patterns used to select the elements you want to style. Selectors can be divided into the following categories:

  • Simple selectors - Select elements based on name, id, class
  • Combinator selectors - Select elements based on a specific relationship between them
  • Pseudo-class selectors - Select elements based on a certain state
  • Pseudo-elements selectors - Select and style a part of an element
  • Attribute selectors - Select elements based on an attribute or attribute value

1. Simple Selectors

Element Selector

The element selector selects HTML elements based on the element name.

Example:

p {
  text-align: center;
  color: red;
}

Selects all <p> elements and styles them with red, centered text.

ID Selector

The id selector uses the id attribute of an HTML element to select a specific element.

Example:

#para1 {
  text-align: center;
  color: red;
}

Selects the element with id="para1" and styles it with red, centered text.

Class Selector

The class selector selects HTML elements with a specific class attribute.

Example:

.center {
  text-align: center;
  color: red;
}

Selects all elements with class="center" and styles them with red, centered text.

Universal Selector

The universal selector (*) selects all HTML elements on the page.

Example:

* {
  margin: 0;
  padding: 0;
}

Removes all default margins and padding from all elements.

Grouping Selector

The grouping selector selects all the HTML elements with the same style definitions.

Example:

h1, h2, p {
  text-align: center;
  color: red;
}

Selects all <h1>, <h2>, and <p> elements and styles them with red, centered text.

2. Combinator Selectors

Descendant Selector (space)

The descendant selector matches all elements that are descendants of a specified element.

Example:

div p {
  background-color: yellow;
}

Selects all <p> elements inside <div> elements.

Child Selector (>)

The child selector selects all elements that are the immediate children of a specified element.

Example:

div > p {
  background-color: yellow;
}

Selects all <p> elements that are immediate children of a <div> element.

Adjacent Sibling Selector (+)

The adjacent sibling selector selects an element that is directly after another specific element.

Example:

div + p {
  background-color: yellow;
}

Selects the first <p> element that is placed immediately after <div> elements.

General Sibling Selector (~)

The general sibling selector selects all elements that are siblings of a specified element.

Example:

div ~ p {
  background-color: yellow;
}

Selects every <p> element that are preceded by a <div> element.

3. Pseudo-class Selectors

A pseudo-class is used to define a special state of an element.

Common Pseudo-classes

Example:

/* unvisited link */
a:link {
  color: #FF0000;
}

/* visited link */
a:visited {
  color: #00FF00;
}

/* mouse over link */
a:hover {
  color: #FF00FF;
}

/* selected link */
a:active {
  color: #0000FF;
}

Styles links in different states.

Structural Pseudo-classes

Example:

/* First child */
p:first-child {
  color: blue;
}

/* Last child */
p:last-child {
  color: red;
}

/* nth child */
p:nth-child(2) {
  color: green;
}

Styles elements based on their position in the document tree.

4. Pseudo-elements Selectors

A CSS pseudo-element is used to style specified parts of an element.

Common Pseudo-elements

Example:

/* First line */
p::first-line {
  color: #ff0000;
  font-variant: small-caps;
}

/* First letter */
p::first-letter {
  color: #ff0000;
  font-size: xx-large;
}

/* Before content */
h1::before {
  content: url(smiley.gif);
}

/* After content */
h1::after {
  content: url(smiley.gif);
}

Styles specific parts of elements.

5. Attribute Selectors

The attribute selector is used to select elements with a specified attribute.

Attribute Selectors Examples

Example:

/* Selects elements with target="_blank" */
a[target] {
  background-color: yellow;
}

/* Selects elements with target="_blank" */
a[target="_blank"] {
  background-color: yellow;
}

/* Selects elements with title containing "flower" */
[title~="flower"] {
  border: 5px solid yellow;
}

/* Selects elements with lang starting with "en" */
[lang|="en"] {
  background: yellow;
}

/* Selects links starting with "https" */
a[href^="https"] {
  background-color: yellow;
}

/* Selects links ending with ".pdf" */
a[href$=".pdf"] {
  background-color: yellow;
}

/* Selects links containing "w3schools" */
a[href*="w3schools"] {
  background-color: yellow;
}

Selects elements based on their attributes and attribute values.

Best Practices

  • Use specific selectors to target elements precisely
  • Avoid using !important unless absolutely necessary
  • Use class selectors for styling and ID selectors for JavaScript hooks
  • Keep your selectors as simple as possible while maintaining specificity
  • Use pseudo-classes for interactive states
  • Group similar styles to reduce code duplication
