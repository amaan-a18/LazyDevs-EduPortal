Introduction to CSS

What is CSS?

CSS stands for Cascading Style Sheets. It is a style sheet language used to describe the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

CSS is one of the core languages of the open web and is standardized across Web browsers according to the W3C specification. Developed in levels, CSS1 is now obsolete, CSS2.1 is a recommendation, and CSS3, now split into smaller modules, is progressing on the standardization track.

Why CSS?

CSS brings style to your web pages by interacting with HTML elements. Elements are the individual HTML components of a web page—for instance a paragraph— which in HTML might look like this: 

<p>This is my paragraph!</p>

If you wanted to make this paragraph appear blue and bold to people viewing your web page through a web browser, you’d use CSS code that looks like this: 

p {
    color: blue;
    font-weight: bold;
}

In the above example, the <p> (paragraph) element is being styled. CSS is added to HTML elements in three main ways:

  • Inline: By using the style attribute inside HTML elements.
  • Internal: By using a <style> block in the <head> section.
  • External: By using an external CSS file linked to the HTML document.

CSS Syntax

A CSS rule consists of a selector and a declaration block. 

selector {
    property: value;
    /* more property-value pairs */
}
  • Selector: Points to the HTML element you want to style.
  • Declaration Block: Contains one or more declarations separated by semicolons.
  • Property: The style attribute you want to change (e.g., color, font-size).
  • Value: Each property has a value that defines how the property should be styled.

CSS Versions

  • CSS1: Released in 1996, basic styling capabilities.
  • CSS2: Released in 1998, added positioning, z-index, and media types.
  • CSS2.1: Refined version of CSS2, became a W3C recommendation.
  • CSS3: Modular approach, introduced animations, transitions, flexbox, grid, and more.

Advantages of CSS

  • Separation of Concerns: Keeps content (HTML) separate from design (CSS).
  • Reusability: Styles can be reused across multiple pages using external stylesheets.
  • Maintainability: Easier to maintain and update styles globally.
  • Page Load Speed: Reduces HTML file size and improves loading performance.
  • Responsive Design: Enables creation of responsive layouts for different devices.

Conclusion

CSS is an essential technology for web development, enabling developers to create visually engaging, responsive, and maintainable websites. Mastering CSS is a fundamental step toward becoming a proficient web developer.
