CSS Box Model

The CSS box model is the foundation of layout design on the web. Every element is considered a rectangular box made up of four areas: content, padding, border, and margin.

Visual Representation

Margin Border Padding Content
Content

1. Content

The innermost part where text, images, or other media appear. Its dimensions are controlled by width and height.

.box {
  width: 200px;
  height: 100px;
}

2. Padding

Space between the content and the border. It inherits the element's background color.

.box {
  padding: 20px; /* all sides */
  padding-top: 10px;
}

3. Border

Wraps the padding and content. Its thickness, style, and color can be customized.

.box {
  border: 5px solid #58a6ff;
}

4. Margin

Space outside the border, separating the element from others. Margins are transparent.

.box {
  margin: 15px; /* all sides */
  margin-left: auto;
  margin-right: auto; /* centering trick */
}

Box-Sizing: border-box

By default, width and height apply only to content. Use box-sizing: border-box to include padding and border in the specified dimensions.

* {
  box-sizing: border-box;
}

Quick Tips

  • Collapsing margins: adjacent vertical margins merge into the larger value.
  • Use outline for debugging without affecting layout.
  • Prefer border-box for predictable sizing.
