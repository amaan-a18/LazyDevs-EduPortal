Master the Flexible Box Layout Module

What is Flexbox?

The Flexible Box Module, usually referred to as flexbox, was designed as a one-dimensional layout model, and as a method that could offer space distribution between items in an interface and powerful alignment capabilities.

When we describe flexbox as being one dimensional, we are describing the fact that flexbox deals with layout in one dimension at a time â€” either as a row or as a column. This can be contrasted with the two-dimensional model of CSS Grid Layout, which controls columns and rows together.

The main idea behind the flex layout is to give the container the ability to alter its items' width/height (and order) to best fill the available space.

Flexbox Basics

Flexbox consists of flex containers and flex items. A flex container is defined using display: flex or display: inline-flex.

Creating a Flex Container

.container {
    display: flex; /* or inline-flex */
}

Flex Direction

The flex-direction property establishes the main axis:

.container {
    flex-direction: row | row-reverse | column | column-reverse;
}

Flex Container Properties

display

Defines a flex container

display: flex
flex-direction

Establishes the main axis

flex-direction: row
flex-wrap

Allows items to wrap

flex-wrap: nowrap
justify-content

Aligns items on the main axis

justify-content: flex-start
align-items

Aligns items on the cross axis

align-items: stretch
align-content

Aligns wrapped lines

align-content: stretch

Flex Item Properties

order

Controls the order of items

order: 0
flex-grow

Defines ability to grow

flex-grow: 0
flex-shrink

Defines ability to shrink

flex-shrink: 1
flex-basis

Sets the initial main size

flex-basis: auto
flex

Shorthand for grow, shrink, basis

flex: 0 1 auto
align-self

Overrides align-items

align-self: auto

Perfect Centering with Flexbox

One of flexbox's most popular features is making centering easy:

.parent {
    display: flex;
    justify-content: center;
    align-items: center;
}

This single rule centers content both horizontally and vertically!

Flexbox vs Grid

While flexbox is one-dimensional (either row or column), CSS Grid is two-dimensional (rows and columns simultaneously).

Use flexbox for:

  • Navigation bars
  • Card layouts in a row/column
  • Centering content
  • Variable width items

Use Grid for:

  • Complex page layouts
  • When you need both rows and columns
  • Precise placement of items
  • Overlapping elements

Browser Support

Flexbox has excellent browser support. All modern browsers support flexbox, including:

  • Chrome 29+
  • Firefox 28+
  • Safari 9+
  • Edge 12+
  • IE 11 (with some bugs)
For older browsers, you might need vendor prefixes or fallbacks.

Common Flexbox Patterns

1. Holy Grail Layout

.container {
    display: flex;
    flex-direction: column;
    min-height: 100vh;
}

.main {
    display: flex;
    flex: 1;
}

.content {
    flex: 1;
}

.sidebar {
    flex: 0 0 200px;
}

2. Card Layout

.cards {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
}

.card {
    flex: 1 1 300px;
}

3. Navigation Bar

.nav {
    display: flex;
    justify-content: space-between;
    align-items: center;
}
