Introduction to HTML

Learn the basics of HyperText Markup Language and start building your first web pages step by step

What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It describes the structure and content of a webpage using a series of elements and tags. HTML tells the browser how to display text, images, links, and other media, forming the backbone of every site you visit.

Unlike programming languages, HTML is declarative: you describe what should appear, not how to compute it. Paired with CSS and JavaScript, HTML becomes the foundation of modern web development.

Basic Structure

Every HTML document follows a simple skeleton. Understanding this boilerplate is your first step toward creating valid, accessible pages:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>

The <!DOCTYPE html> declaration informs the browser to use HTML5 standards. The lang attribute on the <html> tag improves accessibility and SEO by specifying the primary language.

Common Tags

These elements are the building blocks you will use every day. Memorize them early and practice combining them:

  • <h1> - <h6> – Six levels of headings; <h1> is the most important
  • <p> – Paragraph blocks for body text
  • <a> – Anchor (hyperlink); use href to set the destination
  • <img> – Image; requires src and alt attributes
  • <ul>, <ol>, <li> – Unordered lists, ordered lists, and list items
  • <div> – Generic block-level container for layout and styling
  • <span> – Generic inline container for small text fragments

Always close your tags in the correct order to avoid nesting errors that can break layout or accessibility.

Attributes

HTML elements can have attributes that provide extra information. Attributes always come in name/value pairs like class="highlight" or href="https://geeksforgeeks.org". Attributes such as id, class, title, and data-* are invaluable for CSS styling and JavaScript interaction.

Remember to wrap attribute values in double quotes and use lowercase names for consistency and compatibility.

Nesting Elements

Elements can contain other elements, forming a tree-like structure called the DOM (Document Object Model). Proper nesting ensures predictable rendering and cleaner code:

<div class="card">
    <h2>Card Title</h2>
    <p>Some content inside the card.</p>
</div>

Think of nesting like Russian dolls: each element should open and close completely inside its parent. Mis-nesting can cause visual bugs and hinder screen-reader navigation.

Next Steps

Once comfortable with HTML, move on to CSS for styling and JavaScript for interactivity. Practice by building small projects like personal portfolios, blogs, or tribute pages. Explore semantic tags such as <article>, <section>, <nav>, and <footer> to improve accessibility and SEO. Validate your markup with the W3C Validator and learn about responsive design to make your sites look great on every device.
